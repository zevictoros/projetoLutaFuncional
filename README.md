# projetoLutaFuncional

## Motivação

Esse projeto teve como motivação concretizar conhecimentos adquiridos no curso de desenvolvimento web.



## O Que Foi Utilizado

O projetoLuta foi criado com técnicas de HTML, CSS e Javascript.
